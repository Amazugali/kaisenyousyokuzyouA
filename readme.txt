カニスレデスクトップマスコット　VER,1.00

〇せつめい
カニエビはむのSSP用デスクトップマスコットです。
再配布とか改造をしたい場合は下部に書いてある作者ツイッターまでご連絡ください。
ツイッターが死んでいた場合再配布や改造は禁止とします。

〇つかいかた
①http://ssp.shillest.net/からSSP（伺か）という、デスクトップマスコットを動かすためのエンジンをダウンロードします。
②かわいい女の子たちが出て来るので、全部入りのフォルダ（kaisenyousyokuzyou）を女の子にドラック＆ドロップします。これでインストールができました。
※これでインストールできないときはこのファイルをSSP内のghostフォルダに投げ込めばいけるとおもいます。
③右クリックメニューの「ゴースト切り替え」で「カニエビはむ」を選択すれば、いつもの奴らが出てきます。話は少ないですけど勝手にくっちゃべったり時間を教えてくれたりします。邪魔な時は消します。ENJOY。
